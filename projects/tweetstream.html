Trend Analysis on real-time tweet stream data




April 2018 - May 2018

  • Problem Statement

    To capture and analyze real-time tweet streams to understand current popularity of a topic.

    ​
  • Description

    Deployed a pipeline (client-server) for trend analysis over real-time tweet stream using Apache Spark (PySpark) on a local 2-node cluster. A PySpark client uses socket stream and listens to a TCP server, which integrates to Twitter on its behalf and provides the tweets to this socket stream listener. These tweets can be analyzed in real time by accepting a trending term and scouring the tweet stream to count the number of occurrences of the term in each minute.

    ​
  • Tools/Technologies
    Python, tweepy, Apache Spark, PySpark, TCP sockets
