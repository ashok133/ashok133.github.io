Mapping Elevation using Autonomous UAVs in Swarm with Field Acquisition and Resolution Control




Jan 2018 - April 2018

  • Problem Statement

    To develop a formation flying simulation for UAV image acquisition with real-rime control'.

    ​
  • Description

    Led a team of 5 to devise a decentralized algorithm based on relay points for fault tolerant image acquisition by multiple UAVs in swarm formation, simulating algorithm efficiency in both 2D and 3D (Unity). Responsible for building client and server backend (Python, Javascript) and developing a central dashboard for ISRO’s Application Scientists to monitor image mapping missions in phases with real-time control.

    Key Features:

    1. Intelligent autonomous survey and mapping
    2. Minimum human intervention with predictive suggestions related to missions
    3. Ready to be interfaced with publicly available DEMs (CARTOSAT, ISRO)
    4. Estimating flight time, future locations of UAVs and relay points (Helps to evaluate accuracy of planning ahead of time)
    5. Fault tolerance with master-apprentice approach
    6. Rebalancing grid coverage based on flight scenario
    7. Relay points to maximise UAV and server communication range
    8. Intuitive and simple dashboard with extensive statistics and logging

    Applications:
    1. Multilevel imagery for better GIS capabilities that can be integrated with Bhuvan, ISRO
    2. Rescue ops with heat maps (finding humans in landslides, floods)
    3. Forest exploration (counting, classification of trees)
    4. Area grids for controlled fertiliser spray using UAVs over a large field
    5. Surveillance and monitoring system with alarms
    6. Path estimation for constructing roads to reach higher terrains
    7. Intelligent city mapping and target modeling that can be associated with Bhuvan
    8. Phytoplankton analysis over lagoons and other large water bodies
    9. Fish culture study with multilevel imagery using UAVs
    10. River monitoring for cleanliness drives (NMCG - National Mission for Clean Ganga)

    ​
  • Tools/Technologies
    1. 2D simulation - Python, OpenCV
    2. 3D simulation - Unity3D, C#
    3. Dashboard - HTML, CSS, JS, node.js, websockets
    4. Controller engine - Python
    5. Stub Server - Python
