Facial Feature Recognition using Cloud Dataproc




April 2018 - May 2018

  • Problem Statement

    To configure a Cloud Dataproc instance to distribute facial feature mapping among set cluster nodes.

    ​
  • Description

    Configured Apache Spark on Cloud Dataproc to distribute a computationally intensive image processing task onto a cluster of machines. The task involved facial feature mapping using haar cascades over 2 cluster nodes and saving mapped features in GCP buckets. Submitted job detects faces in a set of image files in a custom Google Cloud Storage (GCS) bucket, and writes out image files, with the faces outlined.

    ​
  • Tools/Technologies
    Python, Google Cloud Platform, Cloud Dataproc, Apache Spark, OpenCV
