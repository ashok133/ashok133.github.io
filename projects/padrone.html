Personal Assistance Enabled Drones with Semi-autonomous Flight Capabilities




Feb 2017 - March 2018

  • Problem Statement

    To devise and develop a single software suite platform for semi-autonomy and personal assistant capabilities in a UAV

    ​
  • Description

    The idea of having personal assistant UAVs is inspired by currently available digital assistants on smartphones such as Google Assistant, Siri, Bixby, and Cortana. These assistants are limited to the functionalities of the smartphone and can only perform some tasks that are a part of the flow (of native applications installed). The same idea can be extended to UAV hardware for applications in security, surveillance, asset tracking, agritech, pisciculture tracking, region mapping along with personal use cases such as hands-free photography, navigating, VR streaming, advertisement, etc. Current implementations involve sophisticated approaches that also involve satellites for high altitude imagery. Thus, some of these applications can be approached with drone vehicles, either as standalone assistants or in a swarm. Autonomous flight is being prototyped using computer vision techniques and combining the result with sensor inputs. Drones can be operated using an android app or using voice and gesture inputs. Personal assistance mode works as a stand-alone system to carry out any custom tasks.
    Final Year (Capstone) Project for Bachelor's in Technology, Computer Engineering.

    Project Scope:

    • Command Processing
    • Object Detection
    • Object Tracking
    • Capturing photos on command
    • Image Captioning
    • Region Mapping
  • Tools/Technologies
    1. Companion App - Java, Android SDK
    2. Command Processing Engine - Java
    3. Object Detection & Tracking - Python, OpenCV, AirSim simulator
    4. Image Captioning - Python, Tensorflow (Show and Tell)
    5. Base H/W Platform - PlutoX
